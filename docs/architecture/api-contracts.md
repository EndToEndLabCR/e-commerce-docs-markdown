## Request/Response Format Standards

All API responses will follow this structure:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "per_page": 24,
    "total": 150,
    "total_pages": 7
  },
  "timestamp": "2025-11-12T15:00:00Z"
}
```

Error responses:

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid input data",
    "details": [
      {
        "field": "email",
        "message": "Email is required"
      }
    ]
  },
  "timestamp": "2025-11-12T15:00:00Z"
}
```
